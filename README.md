# Postman Collection – Swagger Petstore API

Этот репозиторий содержит коллекцию Postman для тестирования Swagger Petstore API.

## Содержимое
Коллекция включает запросы для следующих разделов API:
- **Pet** — операции с питомцами (создание, обновление, поиск, удаление)
- **Store** — операции с заказами (создание заказа, получение информации, удаление)

## Источник API
Описание API взято из официальной Swagger (OpenAPI) документации: https://petstore.swagger.io/#/

## Используемые инструменты
- Postman
- Swagger / OpenAPI

## Цель
Практика работы с REST API и написание запросов в Postman.
